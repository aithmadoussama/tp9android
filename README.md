#  LAB 9 : Consommer un Web Service PHP 8 depuis une application Android avec Volley

L’objectif de ce TP est de développer une application Android permettant de saisir les informations d’un formulaire, puis de les envoyer vers un serveur web via une API développée en PHP. Les données seront traitées côté serveur et une réponse sera renvoyée à l’application.

## Technologies utilisées

- Frontend :  
  - HTML / CSS / JavaScript  / java
  - React 

- Backend :  
  - PHP 

- Base de données :  
  - MySQL

- Autres outils :  
  - XAMPP
  - Postman
    
## Fonctionnalités
  - Créer une base de données MySQL pour gérer les étudiants.
  - Développer un Web Service PHP capable d’ajouter et de renvoyer les données en JSON.
  - Construire une application Android qui consomme ce service via Volley.
  - Utiliser Gson pour parser la réponse JSON et afficher les résultats.
